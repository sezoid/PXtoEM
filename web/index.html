О проекте

PXtoEM - простой конвертер, позволяющий с легкостью рассчитать размеры тех или иных объектов при веб-разработке. Благодаря своей простоте, есть 4 версии приложения: веб, расширение для Google Chrome, плагин для Sublime Text 3 и приложение для Android OS.

Установка плагина для Sublime Text 3

  • Скачайте плагин по ссылке выше
  • Если у вас Windows: перейдите и скопируйте файл плагина по следующему пути "C: \ Users \ ИМЯ_ПОЛЬЗОВАТЕЛЯ \ AppData \ Roaming \ Sublime Text 3 \ Installed Packages"
  • После установки в верхнем меню во вкладке "Edit" появится пункт "Px to Em" (есть комбинация клавиш "CTRL + E")
  • Данный пункт выполняет перевод ПИКСЕЛЕЙ В ЭМЫ и не более


Установка расширения для Google Chrome и его запуск

  • Скачайте расширение и сохраните его в удобном для вас месте
  • В Google Chrome идем в меню по пунктам "Дополнительные инструменты - Расширения"
  • В открывшееся окно переносим файл расширения, подтверждаем установку
  • Расширение можно запустить с экрана сервисов или перейдите по URL - chrome://apps/
